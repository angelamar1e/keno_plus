# 🌟 Keno Plus

A mobile application that digitizes the traditional [Keno](https://www.pokernews.com/casino/keno/how-to-play-keno.htm) game, built using Flutter and the BLoC architecture. It includes user authentication, supports betting on multiple tickets simultaneously, offers game history tracking, features quick auto-pick number selection, and simulates wallet operations such as deposits and withdrawals.
